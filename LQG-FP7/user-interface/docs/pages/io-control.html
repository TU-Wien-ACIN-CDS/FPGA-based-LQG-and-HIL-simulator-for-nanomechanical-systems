IO control panel

This persistent panel gives quick access to toggling the feedback output, as well as switching the internal model.

Functionality

Enable feedback

Toggling the switch to the On position enables the feedback output, Off disables it, setting the analog outputs to zero Volts.

Important!: Toggling the switch triggers the Recalculate LQG action, applying any changes made in the Model configuration tab.

Parameter set select

Toggling the switch to the Set 1 position uploads the parameter set 1 to the model, analogously for the Set 2 position.

Important!: Toggling the switch triggers the Recalculate LQG action, applying any changes made in the Model configuration tab.
