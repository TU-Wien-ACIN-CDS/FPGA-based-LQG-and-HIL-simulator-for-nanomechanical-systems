Data Acquisition Tab

In this tab, you can record and plot the internal states, as well as input and output signals.

Functionality

Plot window

Displays the recorded (or simulated) data according to the Channel selection panel.

Right clicking the plot window allows you to adjust the viewport to the current data. Note that this context menu is triggered only when the built-in plot tools are not active (those selected in the toolbar above the window).

Record data

Triggers data recording in the hardware and updates the plot window with Number of frames to plot frames.

Channel record & plot selection panel

Here you can configure which signals you want to record and to plot in order to, for example, save memory, allowing you to record more frames, or to plot only the relevant signals, improving the responsiveness of the GUI. Note that the signals ticked record, but not plot still get recorded into the memory and can be exported later.

Number of frames to record

Configure the number of frames to be recorded after pressing the Record data button. One frame contains all of the channels marked for recording at the same time instance.

Number of frames to plot

Configure the number of frames to be plotted after pressing the Record data button. This value must be smaller than the Number of frames to record.

Recording decimation

This setting allows you to record only every N-th frame. Timing is aligned to the multiples of the LQG sampling rate at 64 ns. Setting the value to 10 results in data captured at the sampling rate of 640 ns.

Reserved RAM

Important: Configure this field to reflect the amount of RAM reserved (in Megabytes) on the specific Red Pitaya interfaced by the GUI.

Output file path

Path to the output Matlab file containing recorded data and parameters after pressing the Save mat file button. Use absolute path to avoid unexpected behaviour. You can right-click the edit field to trigger file explorer dialog.

Note that the filename is amended with a timestamp (yyyy-MM-dd-hh-mm-ss). For example the entry (...)\data\fpgaData.mat creates file fpgaData2026-01-17-11-02-23.mat

Save mat file

Clicking the button exports the currently recorded data including the active parameter set to a Matlab file specified by the Output file path directory. The function exports all of the newest recorded data in memory.

Note: This operation requires fetching a potentially large amount of data from the Red Pitaya and can take a long time.

Plot style switch

Switching the switch switches the style of the plot between linear time domain and log-log power spectral density plot.
