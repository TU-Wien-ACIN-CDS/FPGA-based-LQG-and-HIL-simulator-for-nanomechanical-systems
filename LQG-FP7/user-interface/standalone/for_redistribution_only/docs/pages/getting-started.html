Getting Started

Welcome to the STEMLab LQG7 Documentation. This guide will help you get started.

Quick Start

You can click the topics in the sidebar to get a reference for the content of each tab.

If you did not set up your Red Pitaya yet, follow the steps in the Hardware Setup section.

For more information about the implemented mathematical model, visit the Internal model reference page

Getting it to work

  • Step 1: Connect to the Red Pitaya via terminal, load the bitstream and start the server.
  • Step 2: Navigate the Connect tab in the GUI, fill in the IP address and click connect.
  • Step 3: Make sure to configure the model parameters in the Model Configuration tab.
  • Step 4: Configure the hardware by clicking the Push parameters button.
  • Step 5: For enabling the feedback, click the Enable feedback Switch at the right-hand side in the GUI.
  • Step 6: For recording data, navigate to the Data Acquisition tab and click the Record data button.
