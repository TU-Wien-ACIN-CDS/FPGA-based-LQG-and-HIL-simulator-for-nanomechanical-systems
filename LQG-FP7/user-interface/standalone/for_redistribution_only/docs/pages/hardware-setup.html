Hardware Setup

For setting up the Red Pitaya (RP), refer to the Readme.md at the root of the repository.

Python server

First, make sure you have transferred all the server-relevant files to the RP, including the bitstream.

Configure the RP with the correct bitstream, for example:

echo "lqg_fp7_v1.2.2.bin" > /sys/class/fpga_manager/fpga0/firmware

Then run the server using:

coreconfig/server.py ip_description.yaml

assuming the current working directory is the parent directory of coreconfig and also contains the ip_description.yaml file. The console will display logs of incoming HTTP requests, but you can safely ignore it and now work purely with the GUI. To shut the server down, simply press Ctrl+C while the cursor is active in the console.
