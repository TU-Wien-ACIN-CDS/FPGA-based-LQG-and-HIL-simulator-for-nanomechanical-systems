Internal model reference

Introduction

This implementation allows configuration of a Linear Quadratic Gaussian (LQG) controller with the following parameters:

  • 2-7 internal states - 3 mechanical degrees of freedom (2 states each), 1 detection drift parameter.
  • 1-2 actuation inputs.
  • 1-2 measurement outputs with additive white noise.

Each degree of freedom (DOF) is considered a damped oscillator, defined by a resonance frequency Ω and a damping rate Γ, driven by a Gaussian white noise w. The system is expressed as a differential equation as:

q ¨ ( t ) = − sgn ⁡ ( Ω ) | Ω | 2 q ( t ) − Γ q ˙ ( t ) + c f 1 , q m u 1 ( t ) + c f 2 , q m u 2 ( t ) + 1 m w q ( t ) ,

with q as the displacement in one DOF, cf1/2,q the actuator coupling coefficient for analog output 1 or 2 and the mass of the particle m. In this implementation, all DOFs dynamics are independent and the output function 𝝌 can be any linear combination of all internal states.

In principle, our FPGA implementation can support arbitrary linear discrete-time state space system with 7 states, 2 inputs and 2 outputs, please reach out in case your use case falls outside of what is supported by the GUI.
